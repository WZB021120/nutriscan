# 更新日志

本文件记录 NutriScan 智能营养助手的所有重要更新。

## [1.0.0] - 2026-01-18

### 新增功能
- ✨ **餐食类型智能识别** - 根据时间自动判断早餐(5-10点)、午餐(10-14点)、晚餐(17-21点)或加餐
- ✨ **历史记录查看** - 支持查看过去 7 天的饮食记录，基于 `createdAt` 字段筛选
- ✨ **餐食删除功能** - 支持在日记页面删除不需要的餐食记录
- ✨ **动态用户问候** - Dashboard 根据时间显示"早上好/下午好/晚上好"
- ✨ **动态用户名显示** - Dashboard 和个人中心显示登录用户的真实昵称和头像

### 功能修复
- 🐛 修复 Dashboard 硬编码用户名 "Sarah" 的问题
- 🐛 修复 Diary 页面历史日期无法显示数据的问题
- 🐛 修复 Reports 页面使用模拟随机数据的问题，现在使用真实的历史餐食数据

### 改进优化
- 📈 Reports 页面图表现在基于真实数据渲染，无数据时显示灰色占位
- 📈 健康洞察根据真实平均摄入数据生成建议
- 📈 后端 API 返回 `createdAt` 字段支持历史记录筛选
- 📈 统计数据只计算有记录的天数，避免空数据干扰

### 代码质量
- 🛠 添加 ESLint 配置文件 (`.eslintrc.json`)
- 🛠 添加 `@types/react` 和 `@types/react-dom` 类型定义
- 🛠 添加 `npm run lint` 和 `npm run typecheck` 命令
- 🛠 后端添加 `bcrypt` 依赖准备密码安全升级
